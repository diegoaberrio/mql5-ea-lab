# Inputs

## Riesgo / lotaje
- UseRiskPercent (bool): % riesgo o lotaje fijo
- RiskPercent (double): % de equity a arriesgar
- FixedLots (double): lotaje fijo

## ATR
- ATR_Period (int)
- SL_ATR_Mult (double)
- Trail_ATR_Mult (double)

## Tendencia y S/R
- periodo_ema (int): periodo de SMA (nota: nombre histórico)
- NumeroVelasCalculo (int): ventana para soporte/resistencia
