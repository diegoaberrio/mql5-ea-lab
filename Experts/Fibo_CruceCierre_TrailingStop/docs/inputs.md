# Inputs

## Riesgo / lotaje
- UseRiskPercent (bool): % riesgo o lotaje fijo
- RiskPercent (double): % de equity
- FixedLots (double): lotes fijos

## ATR
- ATR_Period (int)
- SL_ATR_Mult (double)
- Trail_ATR_Mult (double)

## Fibonacci
- NumeroVelasFibo (int): nº de velas para buscar máximo y mínimo
